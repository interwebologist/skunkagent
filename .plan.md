# Feature: duckduckgo search tool

## Goal
Add a free DuckDuckGo-based search tool for the agent using the webfetch to get 
the most up to date docs from the offical api.

## Task list
1. Add a DuckDuckGo tool search tool in tools/ folder. allow checking search with day query. set a default amount of search results , but allow the user nlp to search more results. do a webfetch and search the web about how to implement this.  
2. Wire the tool into agent.py tools to be used my NLP react agent
3. Add logging and error handling, so main react agent can report back the error if it sees one.
4. Add README snippet about tool. use fewest words possible explaining all functionality of the task

## Acceptance criteria
- [ ] Search returns results. Check README on how to do it
- [ ] Tool is callable from FastAPI endpoint which responds from the react agent by query of apply/ endpoint.
- [ ] State updates in stat.txt after each item.
